# Model Name Note

This repository uses the final manuscript name **ECMF-4DD**, which stands for:

```text
Edge-aware Clinical Multi-view Fusion for Disease Diagnosis
```

During internal development, the same model was previously named **ECRD-Net v1.5**. Some source-code class names or historical comments may still contain identifiers such as `ECRDNetV15` for compatibility and traceability.

The name change does not indicate a change in the computational logic. The model architecture, training configuration, and experimental setting used for the reported results remain consistent with the final submitted version of ECMF-4DD.

In the manuscript, README files, and user-facing documentation, the model should be referred to as **ECMF-4DD**.
